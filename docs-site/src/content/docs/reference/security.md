## Reporting a Vulnerability

If you believe you've discovered a security vulnerability in Valkey Admin, please contact the Valkey team at **security@lists.valkey.io**.

Please **do not** create a public GitHub issue for security vulnerabilities. We follow a responsible disclosure procedure — depending on the severity of the issue, we may notify Valkey vendors before releasing a public fix.
